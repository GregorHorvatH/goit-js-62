# Перед началом работы

## Настройка Prettier

- установить плагин Prettier в VS Code
- "editor.formatOnSave": true

## Настройка ESLint

- установить плагин ESLint в VS Code
- "editor.defaultFormatter": "esbenp.prettier-vscode"

## Установка файлов библиотек

- `npm install`
